# Junk Removal Website — Peoria, AZ

A simple, clean static website for a junk removal business serving Peoria, Arizona, designed to attract local search traffic.

## Structure

- `index.html` – Home page (hero, services overview, why choose us, service areas, testimonials, blog teasers)
- `services.html` – Full list of services
- `about.html` – About the business
- `contact.html` – Contact form / quote request
- `blog/` – SEO content articles about junk removal
- `css/style.css` – All styling
- `js/script.js` – Mobile menu + small interactions
- `images/` – Add your photos here (hero image, team photo, etc.)

## How to Use / Publish

This is a static site — no build tools needed.

### Option 1: GitHub Pages (free hosting)
1. Push this repo to GitHub (already done if you're reading this on GitHub).
2. Go to repo **Settings > Pages**.
3. Set source to the `main` branch, root folder.
4. Your site will be live at `https://<username>.github.io/Junk-removal-website/`.

### Option 2: Any web host
Upload all files/folders to your hosting provider's web root (e.g., via FTP or cPanel File Manager).

## Customization Checklist

- Replace placeholder phone number `(623) 555-0123` and email throughout the site.
- Add real photos to the `images/` folder (hero background, about page photo).
- Update the business name "Peoria Junk Removal Pros" if desired.
- Update `sitemap.xml` and canonical URLs if you use a different domain.
- Connect the contact form to a form service (e.g., Formspree, Netlify Forms) since it currently has no backend.
